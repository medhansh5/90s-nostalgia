# ~*~ 90s Nostalgia Homepage ~*~
### *Best viewed in Netscape Navigator 4.0 at 800x600* 🌐

---

## What Is This?

A fully interactive, single-file **Windows 95 desktop experience** built in HTML/CSS/JS — no frameworks, no dependencies, no install. Just pure 90s chaos in one `.html` file.

Double-click it. Watch it boot. Feel the dial-up nostalgia.

---

## Features

| Window | What It Does |
|---|---|
| 🌐 **My Homepage** | Scrolling marquee, rainbow header, hit counter, AIM handle, "under construction" vibes |
| 🎵 **Winamp** | Scrolling track name, animated equalizer bars, 90s playlist, play/skip/shuffle |
| 📖 **Guestbook** | Pre-loaded vintage entries + sign it yourself (A/S/L??) |
| 🔮 **Fortune Teller** | Magic 8-Ball with authentic 90s sass |
| 💥 **Recycle Bin** | Throws a fatal error. Hit "Ignore" to spawn more. |
| 💚 **Matrix Mode** | Start Menu → toggles green rain over the whole desktop |

## How To Use

```
Go to https://medhansh5.github.io/90s-nostalgia/
No download required. No npm install. No vite. Just vibes.
```

### Controls
- **Drag** any window by its title bar
- **Double-click** desktop icons to open windows
- **Taskbar buttons** toggle windows open/closed
- **Start button** (bottom-left) opens the Start Menu
- **Winamp** auto-plays on load — use ⏮ ▶ ⏭ 🔀 to control
- **Fortune Teller** — click the 🎱 or the button to shake

## Customization

Open `90s-nostalgia.html` in any text editor and tweak:

- **Playlist** → find the `tracks` array and swap in your own song names
- **Guestbook entries** → find `initialEntries` to change the pre-loaded messages
- **Fortunes** → find the `fortunes` array and add your own predictions
- **AIM handle / bio** → search for `xX_CoolKid97_Xx` in the welcome window HTML

## Easter Eggs

- Hit **Ignore** on the error dialog 4+ times for a special message 🔥
- The hit counter increments every 15 seconds (someone's always visiting)
- Matrix Mode lives in the **Start Menu** → try it in a dark room
